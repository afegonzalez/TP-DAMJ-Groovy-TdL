# TP-DAMJ-Groovy-TdL

Trabajo Practico para 7531 - Teoría de Lenguaje

## Pre-requisitos
- Groovy latest
- grails 4

## Ejecución local

Una vez clonado el repositorio se debe ejecutar:
```console
$ grails run-app
```
Y dirigirse a localhost:8080

## Ejecuación de las pruebas

```console
grails test-app
```
